комнаты лежат в папках, названия которых соответствуют номеру локации, в которой они находятся.

Направления:
 1
2 0
 3

пример:

780 600 r0 0 0//Комната размером 780 на 600 с названием r0 (название не может быть "NoRoom") с фоном 0 с уровнем освещенности 0
r1 NoRoom NoRoom r3 //справа от комнаты находится комната "r1", сверху и слева пусто, снизу - комната r3. Эти первые две строки обязательны.
Wall 0 0 0 //стена на координатах 0,0 с текстурой №0
Item 30 30 0 //Предмет на координатах 30, 30 с текстурой 0
Spike 30 60 0 //Шип на координатах 30, 60 с текстурой 0
Cannon 60 60 1 1 100 0 //Пушка на координатах 60,60 направленная в сторону 1, с номером текстуры 1, задержкой между выстрелами 10 мс и задержкой начала стрельбы 0 мс
gravityChanger 90 60 0 0 //изменятель гравитации на 90,60 типа 0 с текстурой 0
Decoration 210 454 0 0 0 //декорация на координатах 210, 454 с текстурой 0, свечением 0, на слое 0
